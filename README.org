* db-sample
  | コマンド | 意味                            | 備考                       |
  |----------+---------------------------------+----------------------------|
  | %Q       | 終了(Quit)                      |                            |
  | %C       | 登録件数などの表示(Check)       |                            |
  | %P n     | 先頭からn件表示 (Print)         | n=0: 全件, n<0: 後ろ -n 件 |
  | %R file  | fileから読み込み(Read)          |                            |
  | %W file  | fileへ書き出し(Write)           |                            |
  | %F word  | wordを検索(Find)                | 結果を%Pと同じ形式で表示   |
  | %S n     | データをn番目の項目で整列(Sort) | 表示はしない               |
